# 海豚TV (HaitunTV)

一款功能强大的Android TV直播应用，支持多种电视频道观看，提供丰富的节目指南和个性化设置。

## 功能特点

### 📺 直播功能

- 支持海量电视频道直播
- 高清流畅的播放体验
- 多种播放源选择
- 快速频道切换

### 🎯 频道管理

- 频道分组分类
- 频道收藏功能
- 自定义频道排序
- 频道搜索功能

### 📅 节目指南

- 实时节目信息显示
- 详细的节目预告
- 节目时间表

### ⚙️ 个性化设置

- 解码器选择
- 网络设置优化
- 界面个性化
- 开机启动选项

### 🎤 语音控制

- 支持夏杰语音助手
- 语音频道切换
- 语音节目搜索

### 🌐 网络优化

- 网络状态监控
- 自动网络重连
- 网络速度显示

## 技术架构

### 核心技术栈

- **开发语言**: Java
- **播放引擎**: IjkPlayer
- **图片加载**: Glide
- **网络请求**: OkHttp
- **架构模式**: MVVM + Controller

## 安装说明

### 方法一：直接安装APK

1. 下载最新版本的APK文件
2. 在Android TV设备上安装APK
3. 打开应用开始使用

### 方法二：从源码构建

1. 克隆项目到本地
2. 使用Android Studio打开项目
3. 构建并运行应用

## 使用指南

### 基本操作

- **遥控器操作**：使用方向键导航，确定键选择频道
- **触摸操作**：单击打开菜单，双击打开设置
- **频道切换**：使用遥控器的频道加减键
- **菜单操作**：按菜单键打开主菜单

### 高级功能

- **添加收藏**：长按频道可添加到收藏
- **查看节目指南**：选择频道后可查看详细节目信息
- **网络设置**：在设置中可配置网络优化选项
- **解码器设置**：根据设备性能选择合适的解码器

## 系统要求

- Android 4.4 (API 19) 及以上
- Android TV设备或支持大屏模式的Android设备
- 稳定的网络连接
