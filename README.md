# 海豚TV (HaitunTV)

一款功能强大的Android TV直播应用，支持多种电视频道观看，提供丰富的节目指南和个性化设置。

## 功能特点

### 📺 直播功能 ------------- 🎯 频道管理--------------📅 节目指南

- 支持m3u/txt/gz格式----------频道分组分类-------------实时节目信息显示
- 高清流畅的播放体验------------频道收藏功能--------------自定义节目信息地址
- 多种播放源选择----------------自定义频道排序-----------详细的节目预告
- 快速频道切换------------------内置台标功能--------------每天自动更新

### ⚙️ 个性化设置-------------🎤 语音控制---------------🌐 网络优化

- 软硬解码器选择----------------接入夏杰语音支持---------------网络状态监控
- 个人偏好设置------------------支持语音频道切换---------------自动网络重连
- 界面显示设置-------------------支持直接说出台名--------------网络速度显示
- 开机启动选项-------------------1.2版本以上功能---------------自定义超时切台

源码不含任何片源，可自行在string文件里配置，支持m3u/txt及gz压缩格式。

### 核心技术栈
- **架构模式**: MVVM + Controller
- **开发语言**: Java - **播放引擎**: IjkPlayer - **图片加载**: Glide - **网络请求**: OkHttp

## 系统要求

- Android 4.4 (API 19) 及以上
- Android TV设备或支持大屏模式的Android设备
- 稳定的网络连接
