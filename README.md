# 海豚TV (HaitunTV)

一款功能强大的Android TV直播应用，支持多种电视频道观看，提供丰富的节目指南和个性化设置。

## 功能特点

### 📺 直播功能 ------------- 🎯 频道管理--------------📅 节目指南

- 支持海量电视频道直播----------频道分组分类--------------实时节目信息显示
- 高清流畅的播放体验------------频道收藏功能--------------实时节目信息显示
- 多种播放源选择-----------------自定义频道排序-----------详细的节目预告
- 快速频道切换-------------------频道搜索功能--------------节目时间表

### ⚙️ 个性化设置

- 解码器选择
- 网络设置优化
- 界面个性化
- 开机启动选项

### 🎤 语音控制

- 支持夏杰语音助手
- 语音频道切换
- 语音节目搜索

### 🌐 网络优化

- 网络状态监控
- 自动网络重连
- 网络速度显示

## 技术架构

### 核心技术栈

- **开发语言**: Java
- **播放引擎**: IjkPlayer
- **图片加载**: Glide
- **网络请求**: OkHttp
- **架构模式**: MVVM + Controller

## 系统要求

- Android 4.4 (API 19) 及以上
- Android TV设备或支持大屏模式的Android设备
- 稳定的网络连接
